1) docker run --cpus=.5 ubuntu

2) docker run --memory=100m ubuntu

3) Run commands inside a running container.
	- docker exec {containerId}
	 eg: docker exec 3c7329ea681e ps -eaf
	 
4) If you wish to save data generated by a container then create a volume using
	- docker volume create data_volume
	
	And use this volume while running the container
	- docker run -v data_volume:/var/lib/mysql mysql
